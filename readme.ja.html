概要

小鳥bot は Perl で記述された IRC bot です。プラグインで拡張できます。

動作環境

必須環境

  • perl 5.8.0 以降
  • POE
  • POE::Component::IRC

推奨環境

  • libwww-perl（KotoriBot::Plugin::URIInfo::HTTP 他いろいろなプラグインに必要）
  • Crypt::SSLeay（KotoriBot::Plugin::URIInfo::HTTP で https をサポートする場合などに必要）
  • Image::ExifTool（KotoriBot::Plugin::URIInfo::Image に必要）
  • POE::Component::Client::HTTP（KotoriBot::Plugin::ImasUploaderPolling に必要）
